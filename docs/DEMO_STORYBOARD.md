# 工作台动线

## 视觉元素

- 左侧：带蝴蝶翅膀的小精灵工作台。
- 中间：彩色算法篮子。
- 中下：评分秤。
- 右侧：长期记账本。
- 顶部：当前 best score、覆盖任务数、尝试轮次。

## 动画顺序

1. 小精灵进入网站后持续扇动翅膀。
2. 小精灵从工作台拿起一枚金币。
3. 小精灵带着金币飞到一个算法篮子。
4. 篮子生成候选方案，小精灵带金币飞到评分秤。
5. 小精灵飞起来查看 score，秤显示当前方案分数。
6. 小精灵带结果飞到长期记账本。
7. 如果更优，记账本写入 `+`。
8. 如果没有更优，记账本写入 `-`。
7. best score 面板更新。

## 叙事字幕

```text
Agent 不直接相信任何算法。
它把每个算法当作一个篮子：
试一次、称一次、记一次。
只有分数更低、约束合法、不会超时的方案，才会进入长期记忆。
```
